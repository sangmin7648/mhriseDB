# ⭐ 적용 기술
- CSS 프레임워크 : Bootstrap
- 웹 애플리케이션 프레임워크 : Flask(MVT 모델)
- 웹 서버 : Nginx
- WSGI 서버 : Gunicorn
- DBMS : MariaDB(AWS RDS)
- 서버 환경 : AWS EC2(RHEL -> Ubuntu)
- 웹 크롤링 : Selenium, Beautifulsoup
- 사이트 분석 : Google Analytics
